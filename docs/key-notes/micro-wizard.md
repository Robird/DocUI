
## Micro-Wizard (Draft)

> **Micro-Wizard** （微向导）是一种在LLM驱动的智能体中使用的轻量级、多步骤交互模式。它通过在局部引入动态的、上下文感知的引导步骤，帮助LLM渐进解决局部复杂性，并在完成交互后自动修剪中间状态，以保持上下文的简洁和有效性。

### 例子
如最典型的文本编辑工具`str_replace(string oldText, string newText)`, 传统上在遇到多匹配时会简单报错了事。在经过微向导强化后，再遇到oldText多匹配，会给出一条引导性提示和配套的内容呈现，形如`oldText有3个匹配，分别是：1号...2号...，请选择你想替换的选区序号`，然后动态展现一个选区选择工具接受所提示的选区序号作为输入，接下来如果存在复合你原始意图的选区，你只需输出一个序号，而无需再次复述大段的原始文本和替换文本。当最终编辑完成后，中间这些小波折会从上下文历史中弹出，展现给你的是开头和结尾的结果，结果包含这个小波折的简要印象，以节省掉无后效性的上下文占用。
